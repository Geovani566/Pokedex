# Primeiro contato com consumo de API

Este projeto marca meu primeiro contato com o consumo de uma API utilizando JavaScript.

Foi desenvolvido um mini projeto de Pokédex, consumindo dados da PokéAPI, onde é possível:
- Buscar Pokémon por nome ou número
- Navegar entre Pokémon usando botões de próximo e anterior
- Exibir nome, número e imagem do Pokémon
- Tratar erros quando o Pokémon não é encontrado
- Trabalhar com JavaScript assíncrono (`async/await`)

Tecnologias utilizadas:
- HTML
- CSS
- JavaScript
- PokéAPI
